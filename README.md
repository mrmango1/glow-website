Glow - A beautiful macOS experience built with Astro JS, Tailwind, and React.

## Features

- Comprehensive **Light Mode** and **Dark Mode** support, with automatic detection of system preference
- **Frosted Glass** effect inspired by macOS and Apple Vision Pro
- **Adaptive Layout** for all screen sizes
- **SF Pro** font system for native macOS look
- Built-in patterns for **Search Engine Optimization**
- **GitHub Pages** deployment ready

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [React](https://react.dev/) - UI components
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [TypeScript](https://www.typescriptlang.org/) - Type safety

## Development

### Prerequisites

- [Node.js](https://nodejs.org/) v22+
- [Git](https://git-scm.com/)

### Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/glow-website.git
cd glow-website

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

### Build

```bash
npm run build
```

## Deployment

This project is configured for GitHub Pages deployment. Push to the `main` branch to trigger automatic deployment.
